# Weather Prediction App

A simple Weather Prediction App built using Python, Streamlit, Pandas, Scikit-Learn and Matplotlib.

## Features
- Upload weather data
- Predict future temperature
- Display prediction graph
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-Learn
- Matplotlib

## Run the Project

```bash
streamlit run app.py
```
